## General info
This project is a program for detecting people and counting them. Detection is based on photos and videos.

## Technologies
* opencv-python 4.5.5.64
* imutils 0.5.4
* yolov4-tiny

## Setup
To run this project, install it locally:
```
$ pip install opencv-python
$ pip instal imutils
```
and start program:
```
python run.py
```
